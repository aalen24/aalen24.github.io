Project STORM (Surface Terrain Operations Rover for Mars)

About Project STORM

Project STORM is a project that is run by the Robotics Club of Central Florida, a student run club at UCF. This project's main purpose is to design, prototype, build, and test a rover to participate in the University Rover Challence(URC). URC is an international competition between universities to promote innovation for developing methods to explor the red planet.

Skills from this Project

  • Maintain a systems level understanding of the rover.
  • Delegate tasks to sub-system leads, and follow up on deadlines to maintain progress
  • Manage budget fluctuations/needs.
