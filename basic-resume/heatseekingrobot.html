Heat Seeking Robot

About Heat Seeking Robot

This project was a small group project that was worked on by myself, another computer science student, and a mechanical engineering student. It was part of an introduction to robotics course offered at UCF where students were given the chance to design, build and test a basic robot with the condition that it needed some level of intelligence

Skills from this Project

  • Small team dynamics under pressure of strict deadlines
  • Outlining requirements, goals, and establishing a preliminary project timeline
  • Making adjustments to a plan under changing circumstances
  • Basic embedded programming such as PID control, motor inputs, and sensor data processing
